# RoboType

This was my submission to the 2026 handmade jam. 
I wasn't actually going to enter that jam - I only thought of this idea ~3 days before it ended, but it ended up
being a pretty good one, so I decided to try to finish it as much as possble.

## What is it

It's a typing sim like MonkeyType. Except - you can use Ctrl+W to backspace the last word, among other things.
The input field actually behaves a lot like a normal text input that you might use in other places on your computer.
