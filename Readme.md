# RoboType

This is the completed version of my submission to the 2026 handmade jam. (The one I had by the end of the jam was _not_ this one)

## What is it

It's a typing harness like MonkeyType. 
Except - you can use Ctrl+W to backspace the last word, among other things.
Due to various technical reasons, Ctrl+W will always close the current tab on any web page. 
As such, this program needs to be native.

The input field actually behaves a lot like a normal text input that you might use in other places on your computer.
Try to get the fastest time on every sample. 
Make no mistakes. \[perfect\] !
